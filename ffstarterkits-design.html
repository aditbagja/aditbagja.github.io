Web Design Fathforce Starter Kits Backup & Import Feature

Selama saya magang di PT Inovindo Digital Media, saya mengerjakan desain web dari aplikasi Fathforce Starter Kits. Fathforce Starter Kits merupakan sebuah aplikasi starter untuk sebuah framework yang akan digunakan untuk membangun atau mengembangkan sebuah web aplikasi. Dengan adanya starter kits ini, para penggunanya setidaknya tidak membangun web aplikasi mereka dari awal. Bagian Desain Web yang saya kerjakan adalah bagian Backup dan Import (Wireframe dan Mockup)
